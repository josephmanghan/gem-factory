# Select User - Instructions

## Overview

Present the available user personas from _{{ users.gsheet }}_ and allow the human to select which specific user they want to interact with.

---

## Process

### Step 1: List Available Users

Display the list of all available user personas from _{{ users.gsheet }}_ with their:

- **Name**
- **Age**
- **Occupation**
- **Personality type**
- **Key identifying traits**

### Step 2: Wait for User Selection

Prompt the human to select a user by **name** or **number**.

### Step 3: Confirm Selection

Once a user is selected:

1. **Confirm** the choice
2. **Transition** to that user's persona for all future interactions
